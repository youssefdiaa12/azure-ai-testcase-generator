🤖 AI Test Case Generator

Automated Test Case Generation for Azure DevOps

🎯 What This Project Does

This project is an AI-powered test case generator that automatically creates high-quality manual test cases for your User Stories and publishes them directly to Azure DevOps Test Plans.

Simply write your User Stories with acceptance criteria in Azure Boards, and let Gemini AI generate comprehensive test cases for you automatically!

🔄 How It Works

1️⃣

Fetch User Stories

Pipeline queries Azure Boards for new User Stories with acceptance criteria

2️⃣

Build Hierarchy

Identify parent Epic and Feature to create proper test structure

3️⃣

Generate Test Plans

Create Test Plans and Suites matching your work item hierarchy

4️⃣

AI Generation

Send acceptance criteria to Gemini AI for intelligent test case creation

5️⃣

Create & Link

Publish test cases to Azure DevOps and link back to User Stories
